Title                   : WadSmoosh

Author                  : JP LeBreton

Other Files By Author   : arcadia.wad, oakdoom.wad
Misc. Author Info       : 

Description             : A tool for combining all official id release
                          WADs into a single PK3 file playable with ZDoom
						  and its derivatives. Drop your WADs into the
						  source_wads\ subfolder, then run. A new IWAD
						  file called "doom_complete.pk3" will be
						  generated.
